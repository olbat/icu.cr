    Overview

    Number Formatter

    This class provides a facility for formatting and parsing numbers.

    Usage

    nf = ICU::NumberFormatter.new("en_US")
nf.format(1234.56) # => "1,234.56"
nf.parse("1,234.56") # => 1234.56
