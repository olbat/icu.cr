    class ICU::BreakIterator

    • ICU::BreakIterator
    • Reference
    • Object

    Overview

    This class defines methods for finding the location of -line, sentense and word- boundaries in text.

    Usage

    str = "หน้าแรก"
bi = ICU::BreakIterator.new(str, ICU::BreakIterator::Type::Word)
bi.each { |s| p s }
# => "หน้า"
# => "แรก"
bi.to_a # => ["หน้า", "แรก"]
