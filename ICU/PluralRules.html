    Overview

    Plural Rules

    Provides access to ICU's plural rules functionality.

    Usage

    plural_rules = ICU::PluralRules.new("pl")
plural_rules.select(0) # => "many"
plural_rules.select(1) # => "one"
plural_rules.select(2) # => "few"
